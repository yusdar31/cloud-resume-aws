Membangun Cloud Resume dengan AWS

15 Februari 2026 8 min read

Apa itu Cloud Resume Challenge?

Cloud Resume Challenge adalah project hands-on yang dirancang untuk membantu cloud engineers pemula membangun portfolio yang impressive. Project ini menggabungkan berbagai AWS services dan best practices dalam cloud engineering.

Arsitektur yang Digunakan

Dalam project ini, saya menggunakan beberapa AWS services:

  • Amazon S3 - Hosting static website
  • CloudFront - CDN untuk distribusi global
  • Route 53 - DNS management
  • Lambda - Serverless function untuk visitor counter
  • DynamoDB - Database untuk menyimpan visitor count
  • API Gateway - REST API endpoint
  • Certificate Manager - SSL/TLS certificate

Infrastructure as Code dengan Terraform

Semua infrastructure di-provision menggunakan Terraform. Ini memastikan infrastructure bisa di-reproduce dengan mudah dan mengikuti best practices IaC. 

resource "aws_s3_bucket" "website" {
  bucket = "my-cloud-resume-bucket"
  
  tags = {
    Name        = "Cloud Resume Website"
    Environment = "Production"
  }
}

CI/CD Pipeline

Saya menggunakan GitHub Actions untuk automated deployment. Setiap kali ada push ke main branch, pipeline akan otomatis:

  1. Run tests
  2. Build static files
  3. Deploy ke S3
  4. Invalidate CloudFront cache

Lessons Learned

Project ini mengajarkan saya banyak hal tentang AWS services, infrastructure automation, dan modern DevOps practices. Sangat recommended untuk siapa saja yang ingin memulai karir di cloud engineering!
